This site is a quick way to create a repository of knowledge of sorts. Most pages are basic markdown which should make the real content quite portable going forward.

Wrapping around the markdown is the [Astro framework](https://astro.build/). Within that we are running the [Starlight theme](https://starlight.astro.build/), which provides docs site navigation, styling, and components.

This section of the site is a quick reference to stuff. No real meat and bones, but critical for different scenarios.
