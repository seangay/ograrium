Utilise the ["Arr" suite](https://www.reddit.com/r/Piracy/comments/tlktya/arr_software_suite/) of products to automate the actual downloading and setup into Plex or other products.

## First level implementation

- [Prowlarr](https://github.com/Prowlarr/Prowlarr) - supports management of both Torrent Trackers and Usenet Indexers. It centralises the different feeds for the various other tools in the suite
- [Radarr](https://radarr.video/) - movie collection manager and downloader
- [Sonar](https://sonarr.tv/) - tv collection manager and downloader

## Additional options for setup

- [Overseerr](https://overseerr.dev/) - request management for the suite of products. Useful for the fam to make "requests".
- [Doplarr](https://github.com/kiranshila/Doplarr) - I think this is like the one above.
- [Readarr](https://readarr.com/) - ebook collection manager
- [Bazarr](https://www.bazarr.media/) - handles searching and downloading missing subtitles
- [Lidarr](https://lidarr.audio/) - music collection manager. Not sure if it would be useful given we use spotify at the moment, but worth a look at some point?
